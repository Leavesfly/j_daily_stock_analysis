# 投资组合管理

## 使用场景
当用户询问持仓情况、组合盈亏、仓位管理（如"我的持仓怎么样"、"仓位合理吗"）时使用此技能。

## 执行步骤

1. **获取持仓数据**：从用户提供的持仓信息中提取股票代码和仓位
2. **逐只分析**：对每只持仓股票调用 `get_stock_price` 获取最新行情
3. **组合评估**：计算整体盈亏、仓位分布和风险暴露

## 输出格式

```
## 持仓概览
- 持仓数量：
- 总市值：
- 总盈亏：

## 单只持仓分析
| 股票 | 仓位 | 最新价 | 盈亏 |
|------|------|--------|------|

## 组合评估
- 仓位集中度：
- 行业分布：
- 风险等级：
- 调仓建议：
```
