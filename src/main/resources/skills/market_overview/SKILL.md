# 大盘总览

## 使用场景
当用户询问大盘走势、市场整体情况（如"今天大盘怎么样"、"市场情绪如何"）时使用此技能。

## 执行步骤

1. **获取市场信号灯**：调用 `get_market_overview` 工具，获取市场多空状态和情绪指标
2. **获取大盘历史**：调用 `get_stock_history` 工具（参数 stock_code 设为大盘指数代码），获取指数走势
3. **综合研判**：结合市场信号灯和指数走势，分析市场整体状态

## 输出格式

```
## 市场信号灯
- 多空状态：红灯（风险）/黄灯（谨慎）/绿灯（机会）
- 市场情绪：

## 指数走势
- 近期趋势：
- 成交量变化：

## 市场研判
- 整体判断：
- 操作建议：
```
