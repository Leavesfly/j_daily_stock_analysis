# 策略回测

## 使用场景
当用户要求回测某个交易策略（如"回测均线金叉策略"、"验证一下这个策略的效果"）时使用此技能。

## 执行步骤

1. **获取历史数据**：调用 `get_stock_history` 工具，获取足够长的历史K线数据（建议180天以上）
2. **执行回测**：调用 `run_backtest` 工具，指定股票代码和策略名称
3. **解读结果**：分析回测返回的收益率、最大回撤、胜率等指标

## 支持的策略
- `ma_golden_cross` - 均线金叉策略
- 其他策略可通过策略名称指定

## 输出格式

```
## 回测参数
- 股票代码：
- 策略名称：
- 回测周期：

## 回测结果
- 总收益率：%
- 最大回撤：%
- 胜率：%

## 策略评价
- 策略优劣分析：
- 适用场景：
- 改进建议：
```
