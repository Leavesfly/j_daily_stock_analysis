# 告警管理

## 使用场景
当用户要求设置价格提醒、监控某只股票的关键价位（如"600519跌破1800提醒我"、"设置一个价格告警"）时使用此技能。

## 执行步骤

1. **获取当前行情**：调用 `get_stock_price` 工具，获取股票当前价格作为参考
2. **确认告警条件**：与用户确认告警触发条件（价格上穿/下穿、涨跌幅等）
3. **设置告警**：告知用户告警已记录，将触发条件整理为结构化信息

## 告警条件类型
- 价格上穿：股价突破指定价位
- 价格下穿：股价跌破指定价位
- 涨跌幅告警：日内涨跌幅超过阈值

## 输出格式

```
## 告警设置确认
- 股票代码：
- 当前价格：
- 告警条件：上穿/下穿 ¥xxx
- 告警状态：已设置

## 监控说明
- 系统将持续监控该股票价格
- 触发条件后将通过通知渠道提醒
```
