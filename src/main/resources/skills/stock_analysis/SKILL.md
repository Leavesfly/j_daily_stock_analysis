# 股票综合分析

## 使用场景
当用户要求分析某只股票（如"分析一下600519"、"贵州茅台怎么样"）时使用此技能。

## 执行步骤

1. **获取历史行情**：调用 `get_stock_history` 工具，获取最近30天K线数据
2. **技术指标分析**：调用 `technical_analysis` 工具，获取MA、MACD、RSI、KDJ等技术指标
3. **搜索相关新闻**：调用 `search_news` 工具，获取该股票的最新新闻和舆情
4. **综合分析**：结合技术面和消息面数据，给出分析结论

## 输出格式

```
## 技术面分析
- 趋势判断：（上涨/下跌/震荡）
- 关键指标：MA、MACD、RSI、KDJ 状态
- 支撑/压力位：

## 消息面分析
- 重要新闻及影响评估

## 综合结论
- 交易信号：强烈买入/买入/中性/卖出/强烈卖出
- 评分：0-100
- 操作建议：
- 风险提示：
```
