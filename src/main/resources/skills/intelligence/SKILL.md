# 智能情报

## 使用场景
当用户想了解某只股票的新闻动态、舆情信息（如"最近有什么新闻"、"有什么消息面"）时使用此技能。

## 执行步骤

1. **搜索新闻**：调用 `search_news` 工具，传入股票代码和名称，获取最新相关新闻
2. **分析舆情**：对搜索到的新闻进行情感分析，判断利好/利空
3. **情报汇总**：整理关键信息，评估对股价的潜在影响

## 输出格式

```
## 最新新闻
1. [新闻标题] - 影响评估
2. [新闻标题] - 影响评估
...

## 舆情分析
- 整体情绪：正面/中性/负面
- 关键事件：

## 影响评估
- 短期影响：
- 中长期影响：
- 投资建议：
```
