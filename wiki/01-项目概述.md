# 项目概述

## 简介

**股票智能分析系统 (Daily Stock Analysis)** 是一个基于 Spring Boot 的全功能股票智能分析平台，通过多数据源接入、AI 多智能体协作、技术分析引擎等核心模块，为用户提供自动化的股票分析、决策信号生成、投资组合管理、策略回测等功能。

> 本项目为 Java 实现版本，100% 还原原 Python 版功能。

## 核心能力

| 能力 | 说明 |
|------|------|
| 多模式运行 | Web 服务/API、单次分析、定时调度、大盘复盘、策略回测、干运行 |
| 全覆盖数据源 | 11 个 A 股/港股/美股数据源，支持自动故障切换和熔断恢复 |
| AI Agent 系统 | 6 类专业 Agent + 编排器 + 记忆系统 + 技能系统 |
| 完整 REST API | 13 个 Controller / 12+ 端点，涵盖健康检查、分析触发、行情、组合、告警、回测等 |
| WebUI | 仪表盘、股票分析、投资组合、告警管理、策略回测、AI 问股等 8 大功能页 |
| 多渠道通知 | 13 种通知渠道（企业微信、飞书、Telegram、Discord、邮件等） |
| 策略回测 | 5+ 内置策略（均线金叉、放量突破、牛趋势等） |

## 目录结构

```
src/main/java/com/stock/
├── agent/           # AI Agent 系统（编排器、各类 Agent、工具、技能）
├── api/controller/  # REST API 控制器层（13个Controller）
├── bot/             # Bot 平台集成（命令系统、平台适配）
├── config/          # 应用配置类
├── core/            # 核心分析流水线（Pipeline、上下文构建、结果聚合）
├── dataprovider/    # 数据源适配层（11个数据源实现）
├── llm/             # LLM 服务层（调用、用量追踪）
├── model/           # 数据模型（Entity、Enum、Schema）
├── notification/    # 通知系统（路由、13种渠道发送器）
├── repository/      # 数据访问层
├── scheduler/       # 定时调度
├── service/         # 业务逻辑服务层
├── util/            # 工具类
├── web/             # Web 页面控制器
└── Application.java # 主入口
```

## 运行模式

| 模式 | 命令行参数 | 说明 |
|------|-----------|------|
| 单次分析 | `--stocks 600519,hk00700,AAPL` | 分析指定股票并退出 |
| 定时调度 | `--schedule` | 按 Cron 表达式自动分析 |
| Web 服务 | `--serve` | 启动 API + WebUI（默认） |
| 纯 API | `--serve-only` | 仅启动 API，不触发分析 |
| 大盘复盘 | `--market-review` | 分析主要指数趋势 |
| 策略回测 | `--backtest --backtest-days 180` | 回测指定策略 |
| 干运行 | `--dry-run` | 跳过 LLM 调用，测试流程 |
