# 配置说明

## 配置方式

系统通过 `.env` 文件和环境变量进行配置，优先级:

1. 系统环境变量（最高）
2. `.env` 文件
3. `application.yml` 默认值（最低）

## 基础配置

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `STOCK_LIST` | - | 待分析股票列表，逗号分隔 |
| `MARKET` | `A` | 目标市场: A/HK/US |
| `HISTORY_DAYS` | `60` | 获取历史数据天数 |
| `TIMEZONE` | `Asia/Shanghai` | 时区 |
| `SERVER_PORT` | `8000` | 服务端口 |
| `SERVER_HOST` | `0.0.0.0` | 监听地址 |
| `STORAGE_DIR` | `./data` | 数据存储目录 |

## LLM 配置

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `LLM_MODEL` | `gpt-4o` | LLM 模型名称 |
| `LLM_API_BASE` | `https://api.openai.com/v1` | OpenAI 兼容 API 地址 |
| `LLM_API_KEY` | - | **必填** API 密钥 |
| `LLM_CHANNELS` | - | 备用 LLM 渠道（格式: `model@base_url#api_key`） |
| `AGENT_MODE` | `true` | Agent 模式开关（true/full/false） |

### Agent 模式说明

| 值 | 行为 |
|----|------|
| `true` / `full` | 使用多 Agent 协作模式分析 |
| `false` | 使用传统 LLM 直接分析 |

## 定时调度配置

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `SCHEDULE_CRON` | `0 30 15 * * MON-FRI` | Cron 表达式（默认工作日 15:30） |

### Cron 表达式说明

格式: `秒 分 时 日 月 周`

常用配置:
- `0 30 15 * * MON-FRI` - 工作日 15:30（A 股收盘后）
- `0 0 22 * * MON-FRI` - 工作日 22:00（美股盘前）
- `0 30 9 * * MON-FRI` - 工作日 9:30（A 股开盘）

## application.yml 参考

```yaml
server:
  port: ${SERVER_PORT:8000}
  address: ${SERVER_HOST:0.0.0.0}

spring:
  datasource:
    url: jdbc:sqlite:./data/stock_analysis.db
    driver-class-name: org.sqlite.JDBC
  jpa:
    hibernate:
      ddl-auto: update
    database-platform: org.hibernate.community.dialect.SQLiteDialect
  jackson:
    serialization:
      write-dates-as-timestamps: false
    default-property-inclusion: non_null
    date-format: yyyy-MM-dd HH:mm:ss
    time-zone: Asia/Shanghai

logging:
  level:
    root: INFO
    com.stock: INFO
```
